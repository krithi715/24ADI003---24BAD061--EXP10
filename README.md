This project focuses on building a Movie Recommendation System using Matrix Factorization techniques, specifically Singular Value Decomposition (SVD) and Non-negative Matrix Factorization (NMF), to predict user preferences and generate personalized movie recommendations. The system is developed using the MovieLens 100K dataset available on Kaggle (https://www.kaggle.com/datasets/abhikjha/movielens-100k
), which contains user IDs, movie IDs, and corresponding ratings. The workflow includes loading and preprocessing the dataset, constructing a user-item interaction matrix, handling missing values, and applying matrix factorization methods to learn latent features that represent hidden relationships between users and movies. In the SVD approach, the matrix is decomposed into latent factors, reduced to optimal dimensions, and reconstructed to predict missing ratings, followed by evaluation using RMSE and MAE. In the NMF approach, the rating matrix is factorized into non-negative user and item feature matrices, enabling more interpretable latent representations and recommendation generation, evaluated using RMSE, Precision@K, and Recall@K. Both models are compared based on performance, interpretability, and ability to handle sparsity, along with visualizations such as heatmaps, error plots, and top-N recommendation charts to analyze results effectively.
